Learned Element Tags

  • <h#>: Heading tag. # denotes the size, with 1 being the largest and 6 being the smallest.
  • <br>: Line break. Creates a line break in the text.
  • <hr>: Horizontal Rule. Creates a line across the screen
    • attributes? size="#", where # is the increasing size of the bar
  • <p>: Creates a paragrah of text
  • <em>: Creates emphasis by using italics. Is occasionally used to machine reading. You can still use italics, but they are for specific circumstances. Check the MDN
  • <strong>: Creates a bold emphasis. You can still use a direct <b> tag, however the same conventions as em/i is true
  • LISTS:
    • <ul>: Creates an unordered list. Can be nested. Pass an attribute to change the styling
    • <ol>: Creates an ordered list. Same as above.
  • IMAGES:
    • Called with the tag <img src="", alt="">: Can be web or local links.
    • Alt provides the error text/keywords if the image can't be displayed
  • <a href="">: Anchor tag. Lets you create weblinks
  • TABLES:
    • Created with <able>
      • <tr>: Tags around a table row
      • <td>: Tags around the table data.
      • <thead>: The table header tag
        • <th> Is the table header cell, where you put the header title
          • You can use 'colspan="#"' to make the title take up multiple columns
      • <tfoot>: The table footer tag
      • <tbody>: The table body tag
  • FORMS:
    • Created with <form class="" action="index.html" method="post">
      • enctype="text/plain" will pass regular text thats enetered
      • action denotes what you want to do. You could have mailto:a@b.c and it will open the mail client when submitted
    • You create separate elements of data with the <input type="x" name="" value="">
      • Changing the x variable will chang ethe type of input. Notable examples are:
        • range
        • text
        • password
        • date
        • email
        • month
        • number
        • search
        • submit
        • url
    • Each of these input boxes are labelled with <label for="">title here</label>
    • there is an additional larger/resizable textbox called with <textarea name="Message" rows="x" cols="y">
