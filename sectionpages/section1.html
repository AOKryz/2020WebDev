Introduction

  • Websites are structured into three separate parts:
    • HTML files, which provide the structure of the website. Content and text goes here
    • CSS files, which are the stylistic framework for the site. Formatting, alignment and the like
    • JS files: Javascript files which provide the functionality of the website
  • The general steps for solving a problem are:
    1. Examine EvR: What where your expectations, and what was the actual result.
    2. Step through the debugger: In a systematic manner, step through the code to see where you have a EvR deviation
    3. Search resources for idential or similiar problems: Stackoverflow, google, MDN
    4. Compare your code to Verified code: Find things that are similiar and compare how someone else did it compared to how you did it.
    5. debug console: Look for errors and try to interpret them. Search the errors in your resources
    6. Ask for help: Ask online on stack, reddit, or a discord you frequent. Make sure you provide:
      1. Your expected results
      2. Your actual results
      3. your code
      4. your console output
