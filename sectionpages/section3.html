  3. What is deprecation?
    • The process of use stoppage for commands, or elements, that have been replaced by better, newer or more advanced features. An example are all the attributes for the table element. They have all been replaced by modification through CSS
  4. Why are forms useful?

  • TABLES
    • Used to format data in a visually pleasing, and customized way
    • Nested inbetween <table></table> tags.
    • Tables can also have Headers, Footers and Body just like the html tag.
      • headers are useful for titles. remember to use to <th> to input the header data. However, <th> can be called from the body, but will be styled the same as the rest of the table.
  • FORMS
    • Used to get user input. Have a very large variety of types of possible input.
